# Himanshu Verma - Premium Modern Portfolio

A premium, modern, and high-performance portfolio website built with Next.js (Pages Router), Tailwind CSS v4, and Framer Motion.

## Features

- **Premium Design**: Dark theme with glassmorphism, neon highlights, and modern typography.
- **Advanced Animations**: Smooth page transitions, scroll reveals, floating tech badges, and interactive cards.
- **Dynamic Projects**: Portfolio listing with category filters and detailed project case studies.
- **Interactive Effects**: Custom glowing cursor, mouse spotlight background, and tilt-effect cards.
- **SEO Optimized**: Meta tags for all pages, high-performance image handling, and accessible semantic HTML.
- **Responsive**: Mobile-first design that looks stunning on all devices.

## Tech Stack

- **Framework**: Next.js (Pages Router)
- **Styling**: Tailwind CSS v4 (CSS-based theme configuration)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Slider**: Swiper.js
- **Utils**: clsx, tailwind-merge

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Customization

All the content of the website is managed through the data files in `src/data/`:

- **Profile & Socials**: `src/data/profile.js`
- **Skills**: `src/data/skills.js`
- **Projects**: `src/data/projects.js`
- **Services**: `src/data/services.js`
- **Experience**: `src/data/experience.js`
- **Education**: `src/data/education.js`
- **Blogs**: `src/data/blogs.js`
- **Testimonials**: `src/data/testimonials.js`

## Deployment

This project is optimized for deployment on **Vercel**. Simply connect your GitHub repository and Vercel will automatically detect the Next.js settings.
